## Что делает проект?
Демонстрирует метод фрагментации TLS-пакетов для изучения работы систем Deep Packet Inspection (DPI).

## Как это работает?
1. Мониторит доступность ресурсов
2. При блокировке запускает локальный прокси-сервер
3. Разбивает TLS ClientHello на мелкие фрагменты с задержками
4. DPI не может собрать полный паттерн → соединение проходит

## Запуск
`python3 dpi_bypass.py`
